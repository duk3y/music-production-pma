## Music Review Application

# Steps to building project
1. Clone repo
2. Create a new virtual enviroment by running `python -m venv ./venv` and then activating it.  For MacOS, this is `source ./venv/bin/activate`.  For Windows, it is `.\venv\Scripts\activate`.
3. Run "pip install -r requirements.txt" to install the correct dependencies.
